# vue

## 简介

1. vue 是 js 的一个框架

vue 核心: vue-core + components + vue-router + vuex + vue-cli

渐进性: 就是在上述核心中可以使用一个或多个来写项目，任意的搭配技术栈

2. mvvm 

vue 很早之前都是很多的思想都是从后端移植过来的


比如 mvc
    view           controller            model
     |
    view        viewmode          model
    视图         视图模型          数据
